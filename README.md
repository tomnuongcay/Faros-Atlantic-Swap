# Faros-Atlantic-Swap
Faros atlantic swap bot



# Pre-Requirements


 - phrs ( on atlantic chain )


# Clone the repo


 git clone https://github.com/BamarAirdropGroup/Faros-Atlantic-Swap.git && cd Faros-Atlantic-Swap && npm install



# add private key in .env ( add PRIVATE_KEY_1/2...)

 nano .env



# add proxy if you wish ( username:password@ip:port  )

 nano proxy.txt


# Running 

 npm start
